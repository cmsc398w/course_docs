# System Monitoring Dashboard Project

## Overview

In this project, imagine you are a software engineer / sysadmin at a company and you get 3 customer tickets triaged to you. It's your job to read the ticket and diagnose the issue by using common Linux networking tools. See the details on how you should reproduce the tickets in their respective folders.

## Requirements

### Write Up (10 points each x 2 tickets)

Pick 2 out of the 3 tickets to complete. If you do all 3 tickets, that will be a 5 point bonus.

For each ticket, you must write up a one paragraph (7 sentences max) bug report that contains the following:

- A brief description of the issue (2 points)
- Steps you took to reproduce the issue (2 points)
- What tools you used to diagnose the issue, and how did you use them (2 points)
- What the issue is and how you fixed it (2 points)
- 2 final points will be given for the correct diagnosis of the issue.

See more information in the respective ticket folders.
