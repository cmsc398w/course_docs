** What is the Shell?

***                                                          :B_definition:
:PROPERTIES:
:BEAMER_ENV: definition
:END:

A program that interprets commands and interfaces with your operating system.

*** Key Point
:PROPERTIES:
:BEAMER_ENV: block
:BEAMER_ACT: <2->
:END:

Requires a *terminal* to interact with the shell

** Terminology

*** Two Key Concepts
:PROPERTIES:
:BEAMER_ENV: block
:END:

*Terminal:* The interface program (iTerm2, Windows Terminal, etc.)

*Shell:* The command interpreter (bash, zsh, fish)

*** Relationship
:PROPERTIES:
:BEAMER_ENV: alertblock
:BEAMER_ACT: <2->
:END:

Terminal runs the shell → Shell executes your commands

** The Prompt

*** Anatomy of the Prompt
:PROPERTIES:
:BEAMER_ENV: block
:END:

=username@domain directory$=

*** Components
:PROPERTIES:
:BEAMER_COL: 1.0
:END:

| Component   | Description                      |
|-------------+----------------------------------|
| username    | Currently logged in user         |
| @           | Separator (convention)           |
| domain      | Machine name                     |
| directory   | Current working directory        |
| $           | User-level shell indicator       |
| =#=         | Root-level shell indicator       |

** What is xargs?

***                                                          :B_definition:

** Practical Example: Auto-Backup
:PROPERTIES:
:BEAMER_OPT: fragile
:END:

*** A Robust Backup Script
:PROPERTIES:
:BEAMER_ENV: block
:END:

#+BEGIN_SRC bash
#!/bin/bash
set -euo pipefail

DEST="/backups/$(date +%Y-%m-%d)"
mkdir -p "$DEST"

for dir in "$@"; do
  if [ -d "$dir" ]; then
    echo "Backing up $dir..."
    tar -czf "$DEST/$(basename "$dir").tar.gz" "$dir"
  else
    echo "Warning: $dir is not a directory" >&2
  fi
done
#+END_SRC

** Analysis: Auto-Backup
:PROPERTIES:
:BEAMER_OPT: fragile
:END:

*** Why this is "Robust"
:PROPERTIES:
:BEAMER_ENV: exampleblock
:END:

1. *Safety:* =set -euo pipefail= ensures the script stops if a directory can't be created or a command fails.
2. *Dynamic:* Uses =$@= to process any number of folders passed as arguments.
3. *Error Handling:* Redirects warnings to =stderr= (=>&2=) so they don't pollute the standard output.
4. *Clean Paths:* Uses =basename= to ensure the archive name is clean even if a full path is provided.
